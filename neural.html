Rede Neural

Estou progredindo no desenvolvimento de um projeto de rede neural, explorando os conceitosde aprendizado de máquina para criar um modelo de previsão de demanda de mercado.

Apesar de ser um desenvolvedor júnior em Python, estou entusiasmado com esse projeto! A rede aprende com dados em tempo real, vinculado a api do banco de dados.

Objetivos:

* Taxa de previsão acima de 80%

* Aprendizado em ciclo de 100 Epochs

* Sistema de loss

Considero esse projeto, como uma oportunidade de me aprofundar em Machine learn.
